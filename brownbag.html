Turning Experiments On

Type chrome://flags into Chrome's URL bar

Search for "Enable Developer Tools experiments"

Click "Enable"

Restart Chrome

Enabling Frameworks Debugging (aka skip third parties)

Make sure experimental dev tools are on

Open Developer Tools (F12 or right click and inspect element)

Go to "General" tab

Under "Sources" toggle "Skip stepping through sources with particular names"

Add third party library names separated by "|" (example: "jquery|yui|event-base")

WARNING sometimes you want to go in the libraries so remember to turn this off if so

Stringify Command

JSON.stringify(x, null, 4)

x: JSON object you want to format

null: filter you want to use (I never use so not familiar with this)

4: the amount of spacing you want

This will still be hard to read in the browser so wrap with copy() command to use in your favorite editor

copy(JSON.stringify(x, null, 4))
